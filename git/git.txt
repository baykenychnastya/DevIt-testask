Ви закінчили роботу над завданням для клієнта і ви маєте робочий код,
який був створений без використання git.
Клієнт створив порожній репозиторій git@example.com:example/test.git і розшарив вам доступ.

-----Завдання-----
Опишіть ваші подальші дії щоб ваш код з'явився у гілці master в репозиторії git@example.com:example/test.git

----Для того, щоб завантажити ваш код у репозиторій на сервері git@example.com:example/test.git, необхідно виконати такі дії:-----
1. Встановіть Git (якщо у вас ще немає).
Ви можете завантажити його з офіційного сайту Git: https://git-scm.com/downloads.
2. Відкрийте командний рядок.
3. Перейдіть до папки з кодом, який ви хочете завантажити в репозиторій.
4. Ініціалізуйте Git репозиторій локально
--------------git init---------------------------
5. Додайте всі файли та зміни до комміту:
--------------git add .--------------------------
6. Зробіть коміт із вашими змінами:
--------------git commit -m "Перший коміт"-------
7. Вкажіть репозиторій як джерело:
--------------git remote add origin git@example.com:example/test.git----------
8. Завантажте ваш код у у гілку master:
--------------git push -u origin master----------
